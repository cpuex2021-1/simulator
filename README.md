# simulator
## ビルド方法

```$ make```

## 実行方法

```$ ./simulator```

## 説明

***実行方法が変わりました***

***コマンドの短縮形も追加され、run（最初から実行）とcontinue（続きから実行）が区別されました。`help`で確認してみてください***

*アセンブリとバイナリを読むコマンド(read/eat)がそれぞれ追加されました。シミュレータを立ち上げてからファイルを読み込んでください*

FPUはC++標準で対応

はじめてつかうときはhelpとタイプすると使い方が表示されます

```$ make```
